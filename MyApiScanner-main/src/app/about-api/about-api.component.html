Movie Database Alternative
> À propos

Vous cherchez une API pour fournir des données sur vos films préférés ? Consultez l'API Alternative pour enrichir votre application. L'API obtient des informations de la base de données et renvoie les résultats au format JSON ou XML. Vous pouvez même effectuer une recherche par IMDb ID pour obtenir des résultats similaires à partir de la base de données. Consultez les points de terminaison de l'API pour interroger différents types de données, notamment:

  • - Des listes de titres de films
  • - Affiches de films
  • - Episodes de séries télévisées et bien plus encore !
